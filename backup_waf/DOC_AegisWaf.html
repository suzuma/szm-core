Class Waf

Sistema de seguridad perimetral para la detección de intrusiones, análisis de comportamiento y bloqueo de bots de IA. Implementa una arquitectura de seguridad por capas para mitigar ataques antes de que alcancen la lógica de negocio.

Autor: Mtro. Noe Cazarez Camargo. SuZuMa

Propiedades Críticas

protected array $rules = ['sql_injection', 'xss', ...]

Reglas de inspección de payload. Lista de funciones que se ejecutan dinámicamente para detectar patrones de ataque en parámetros GET y POST.

Documentación de Métodos

protected function sql_injection($v): bool

Motor de detección de SQL Injection. Identifica patrones de UNION SELECT, ataques lógicos (OR 1=1), comentarios y funciones de tiempo (SLEEP/BENCHMARK).

Parámetro Tipo Descripción
$v mixed El valor a evaluar.
Retorna bool True si se detecta un patrón malicioso.

protected function normalize($value): string

Normaliza una entrada para facilitar la detección de payloads maliciosos. Maneja URL encoding múltiple (hasta 3 niveles), HTML entities y remoción de Null Bytes para evitar evasión de filtros.

Parámetro Tipo Descripción
$value mixed Valor de entrada (string o array).
Retorna string Valor normalizado listo para análisis.

protected function analyzeClientReputation(): void

Analiza la reputación del cliente basándose en la integridad de los encabezados HTTP y en la identificación del User-Agent. Detecta Fake Browsers, bots de programación (Curl, Python) y scrapers de IA/SEO.

protected function detectCloudAttack(): void

Detecta si una petición proviene de infraestructura Cloud (AWS, DigitalOcean, etc.). Utiliza una estrategia en tres capas: rangos IP binarios, historial de riesgo y Reverse DNS como fallback.

protected function detectInferencePatterns(): void

Detección de "Inference Jump". Identifica clientes que acceden directamente a APIs internas o endpoints profundos sin haber pasado por las páginas de entrada normales (Home o Login).

protected function rateLimit(): void

Aplica control de Rate Limiting por dirección IP. Utiliza Redis (INCR atómico) como mecanismo principal y MySQL como fallback. El límite actual es de 60 peticiones por minuto.

protected function xss($v): int|false

Detecta intentos de Cross-Site Scripting (XSS). Busca etiquetas script, protocolos javascript:, atributos de eventos (onerror, onload) y acceso a cookies.

protected function detectFuzzing(): void

Detecta ataques de fuzzing basados en la frecuencia de activación de reglas de seguridad. Si una IP/Fingerprint activa más de 10 alertas en 2 minutos, se suma un puntaje de riesgo agresivo.

protected function getIpDetails(string $ip): array

Obtiene información geográfica e ISP de una IP. Implementa caché en Redis por 24 horas y un timeout de 1 segundo para no penalizar el rendimiento del WAF.
